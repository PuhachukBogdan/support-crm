# reference/ — внешние входы (НЕ наша истина)

> Здесь лежат **внешние материалы**, которые мы **оцениваем, но не считаем авторитетными**. Это вход для наших решений, а не источник правды. Источник правды — `../decisions/` (ADR) и наше видение. При конфликте наши решения всегда важнее.

| Файл | Что это | Как относиться |
|------|---------|----------------|
| `pm-roadmap-notes.md` | Разбор поверхностного роадмапа ПМ («Zendesk 2.0»): прямые конфликты с нашими решениями + новое с пометкой «вероятно». | Гипотезы к обсуждению. План/решения **не меняем**, пока сами не признаем improvement. |
| `chatwoot-security-report.md` | Внешний security-обзор Chatwoot (исходный отчёт, был в корне репо). | Вход для `../security/findings.md`, где 47 находок переоценены под нас. **Не** абсолютная истина. |

Наш синтез этих входов живёт в рабочих доках: `../security/findings.md`, `../zendesk-findings.md`, `../decisions/`.
